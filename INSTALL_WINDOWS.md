# Windows Installation - AuroriaLink

## 💻 Windows Installation

> **Warning:** Your antivirus (including Windows Defender) may display an alert. This is normal behavior for applications that are not certified by Microsoft. The file is safe.

---

## 🚀 Download

Download the **`windows.zip`** archive from [the releases page](https://github.com/Sebastien-VZN/auroria_link/releases).

---

## 📋 Installation Steps

### Step 1: Extract the Archive

1. **Download the `windows.zip` archive**.
2. **Right-click** on the downloaded file.
3. Select **Extract All...** or use software like WinRAR, 7-Zip, etc.
4. **Choose a location** to extract the files.

### Step 2: Launch the Application

1. **Open the extracted folder**.
2. **Double-click on the `.exe` file** (for example: `aurorialink.exe`).

### Step 3: Bypass Windows SmartScreen (if necessary)

If a blue window **"Windows protected your PC"** appears:

1. Click on the **More info** link.
2. Then click on the **Run anyway** button.

### Step 4: Allow the Application in Your Antivirus (if necessary)

If your antivirus quarantines the file:

1. **Open your antivirus settings**.
2. **Create an exception** to allow the `aurorialink.exe` file.
3. **Restore the file** from quarantine if necessary.

---

## ✅ Done!

The application is now ready to use on Windows 10 and later versions.

---

## ⚠️ Why These Security Alerts?

These alerts are normal for applications that are not certified by Microsoft (paid certification). **The file is completely safe and contains no malware.**

### Code Signing Certificate

AuroriaLink is not signed with a Microsoft certificate (high annual cost). This is why Windows SmartScreen and some antivirus software may display warnings.

**You can use the application with confidence.**

---

## 🔧 Troubleshooting

### The Application Won't Start

- Verify that you are using **Windows 10 or later**.
- Make sure the entire folder has been extracted (not just the executable).
- Try launching the application **as administrator** (right-click → Run as administrator).

### Antivirus Constantly Blocks the Application

- Add the AuroriaLink folder to your antivirus **exclusions**.
- Temporarily disable real-time protection during installation, then re-enable it.
